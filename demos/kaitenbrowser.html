Welcome to Kaiten Browser
This website is a demo of Kaiten, a new navigation model for web applications that allows you to browse thru one or several applications in a single continuous space. Watch the video.

Kaiten is a new navigation model for web applications. Unlike conventional applications where each screen replace another, Kaiten stacks the screens side by side horizontally, like columns, and provides a continuous user experience. More about Kaiten…

Kaiten takes its name from kaiten-zushi, a sushi restaurant where the sushi are placed on a rotating conveyor belt. Like the conveyor belt, Kaiten moves the screens on a continuous infinite space.

For a seamless experience, download the KaitenBrowser Extension for Firefox!

Kaiten is a free software written in javascript. You can download Kaiten jQuery plug-in and create apps and connectors to existing apps.

Kaiten is a component of Officity, an open source web applications platform. More about Officity…

Kaiten and Officity are products of Nectil, a company based in Brussels specialized in ad-hoc management applications. More information about Nectil…

For this demo, we had integrated only basic features of Wikipedia, Twitter, and Crunchbase. Kaiten is open source and anybody can create connectors. Download Kaiten, create your own connectors, and send us your feedbacks!

Bienvenue sur Kaiten Browser
Ce site est une démonstration de ce que peut faire Kaiten, un nouveau modèle de navigation pour applications web qui vous permet de naviguer dans un espace continu sur une même application ou au travers de plusieurs applications. Visionner la video.

Kaiten est un nouveau modèle de navigation pour des applications web. Contrairement aux applications conventionnelles où chaque écran remplace le précédent, Kaiten empile les écrans côte à côte horizontalement, comme des colonnes, et offre à l'utilisateur une expérience de navigation tout en continuité. Plus d'info sur Kaiten…

Kaiten tire son nom de kaiten-zushi, le restaurant où les sushis sont présentés sur un tapis roulant continu. Tout comme le tapis roulant, Kaiten présente les écrans en les bougeant sur un espace continu et infini.

Pour une utilisation intégrée, téléchargez l'extension KaitenBrowser pour Firefox !

Kaiten est un logiciel libre écrit en Javascript. Vous pouvez télécharger Kaiten sous la forme d'un plug-in jQuery et créer vos propres applications et connecteurs.

Kaiten est un composant d'Officity, une plateforme d'applications web open source. Plus d'information sur Officity…

Kaiten et Officity sont des logiciels de Nectil, une société basée à Bruxelles, spécialisée dans la création d'applications de gestion sur mesure. Plus d'information sur Nectil…

Pour cette démonstration, nous n'avons intégré que les fonctionnalités de base de Wikipedia, Twitter, et Crunchbase. Kaiten est open source et tout le monde peut créer des connecteurs. Téléchargez Kaiten, créez vos connecteurs, et envoyez-nous vos impressions!
